# My account app via jwt-auth

## Installed dependencies
### Server:
json-server 
json-server-auth

### Client:
axios
@reduxjs/toolkit
react-redux
react-router-dom@6
tailwindcss
postcss
autoprefixer
@headlessui/react
@heroicons/react
@tailwindcss/forms

## Local launching of the project:
node version - 16.15.1

### client: >cd client 
    npm install
    npm start (PORT 3000);
### server: >cd server
    npm install
    npm run server (PORT 5000);
